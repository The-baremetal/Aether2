Aether Programming Language for Windows
======================================

Installation:
1. Extract the archive to C:\Program Files\Aether
2. Add C:\Program Files\Aether\bin to your PATH environment variable
3. Open Command Prompt and run: aether --help

Alternative:
- Use the included aether.bat file to run Aether without modifying PATH
- Double-click aether.bat or run it from Command Prompt

Directory Structure:
- bin\aether.exe - Main executable
- packages\ - Standard library packages
- aether.bat - Convenience script

For more information, visit: https://github.com/The-baremetal/Aether2
