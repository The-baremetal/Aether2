Aether Programming Language for NetBSD
======================================

Installation:
1. Extract the archive: tar -xvf aether-netbsd_amd64.tar
2. Copy files to system: sudo cp -r usr/local/* /usr/local/
3. Run: aether --help

The binary will be installed to: /usr/local/aether/bin/aether
A symlink will be created at: /usr/local/bin/aether

Uninstallation:
Run: sudo rm -rf /usr/local/aether
Run: sudo rm -f /usr/local/bin/aether

For more information, visit: https://github.com/The-baremetal/Aether2
